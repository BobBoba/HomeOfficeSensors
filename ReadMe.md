## Temperature measurment device with display

It uses:
- ESP8266 chip
- AM2302 for indor usage
- AM2301 for outdoor
- LCD screen 16x2 text
- 220->5v power source

It updates screen every second.

Every minute it send telemetry to Azure IoT Hub.
